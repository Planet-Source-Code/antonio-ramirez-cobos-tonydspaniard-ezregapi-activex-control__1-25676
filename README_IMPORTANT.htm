EzRegAPI Test-Application Setup  
 
Extract project files from the ZIP file to desired folder and open regapi.vbg. This project group file will open both, the ActiveX Control and the test application.

Some of the features of the controls such as the association of specific file types, do not work on Windows NT or 2000. It can be easily changed, but I don't have the time to work more with this control [well, may be if it has any commercial use... let me know anyway at TonyDSpaniard@hotmail.com]

The EzRegAPI.chm help file included it is not complete nor free of ERRATA. I didn't had the time to update it yet, please forgive me and make sure you follow the examples on the test application. The rest of the features not documented on the help file, will have to be understood with your trial-and-error expertise [welcome to my world].
